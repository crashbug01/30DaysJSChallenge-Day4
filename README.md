## About This Project

![GitHub last commit](https://img.shields.io/github/last-commit/crashbug01/30DaysJSChallenge-Day4)<br>

This project is a deep dive into JavaScript fundamentals, specifically focusing on advanced array manipulation. Using a dataset of famous inventors and people, the script performs complex data filtering, sorting, and aggregation without the use of external libraries. This project is part of the "30 Days of JavaScript" challenge by Wes Bos.

### Key Learning Objective

- Filtering Data: Using .filter() to create sub-arrays based on specific conditions, such as inventors born within a certain century.

- Mapping Arrays: Using .map() to extract specific properties from objects—like combining first and last names—and returning a new array of the same length.

- Sorting Logic: Implementing .sort() with custom comparator functions to arrange data chronologically by birthdate or alphabetically by strings.

- Data Reduction: Utilizing .reduce() for powerful data aggregation, such as calculating the total lifespan of all inventors or counting the frequency of occurrences within an array.
